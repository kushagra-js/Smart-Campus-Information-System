# Smart Campus Information System

A Python-based Smart Campus Information System developed as part of the Python Programming Laboratory.

## Features
- Student Registration and Grade Evaluation
- Course Enrollment Management
- Student Record Management
- Event Participation Analysis using Sets
- Sorting and Searching of Student IDs
- Fee Calculation
- File Handling and Report Generation
- Student Performance Analytics using NumPy, Pandas and Matplotlib

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
